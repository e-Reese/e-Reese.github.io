Building a Travel Router with Raspberry Pi

By Ethan Reese August 7, 2025

My complete travel router setup using a Raspberry Pi 4

My purpose for building this travel router was to solve the problem of hotel Wi-Fi that limits connections to one device and to create a secure network while traveling. In this tutorial, I'll walk you through how I built a portable travel router using a Raspberry Pi that solved these problems.

What You'll Need

  • A laptop or desktop computer
  • Raspberry Pi (I used a Pi 4, but Pi 3B+ works too)
  • MicroSD card (16GB or larger recommended)
  • USB Wi-Fi adapter (for dual Wi-Fi capability)
  • Portable power bank (optional but recommended)
  • Small case for protection
  • Ethernet cable (optional)
  • A monitor, keyboard, and mouse (optional)

Raspberry Pi OS Setup

Step 1: Download Raspberry Pi Imager

First, download the Raspberry Pi Imager from the official Raspberry Pi site.

Once downloaded, open the imager and select the Raspberry Pi OS (64-bit) image. Select the SD card and click "Write" to flash the image to the SD card.

Raspberry Pi Imager Settings

# Flash the image to your SD card
sudo dd if=RaspAP-bcm27xx-bcm2711-rpi-4-squashfs-factory.img of=/dev/sdX bs=4M conv=fsync
# Replace /dev/sdX with your SD card device

After flashing, insert the SD card into your Raspberry Pi and connect it to your computer via Ethernet.

Software Setup

We'll be using RaspAP as our router operating system.

Step 1: Download and Flash RaspAP

First, download the appropriate RaspAP image for your Raspberry Pi model from the official RaspAP site. 

# Flash the image to your SD card
sudo dd if=RaspAP-bcm27xx-bcm2711-rpi-4-squashfs-factory.img of=/dev/sdX bs=4M conv=fsync
# Replace /dev/sdX with your SD card device

After flashing, insert the SD card into your Raspberry Pi and connect it to your computer via Ethernet.

Basic Configuration

Once RaspAP is running, you'll need to access the web interface to configure it.

Step 2: Access the Web Interface

Connect to your Raspberry Pi using SSH or by navigating to 192.168.1.1 in your web browser. 

ssh root@192.168.1.1
# Default password is usually blank

Wi-Fi Configuration

Now we'll set up the dual Wi-Fi configuration - one interface to connect to an existing network, and one to create our own network.

Step 3: Configure Wi-Fi Interfaces

In the RaspAP interface, navigate to Network → Wireless and configure both interfaces:

  • The built-in Wi-Fi: Set as an access point (creates your network)
  • USB Wi-Fi adapter: Set as a client (connects to hotel/external Wi-Fi)
# Example configuration in /etc/config/wireless
config wifi-device 'radio0'
    option type 'mac80211'
    option channel '36'
    option hwmode '11a'
    option path 'platform/soc/fe300000.mmcnr/mmc_host/mmc1/mmc1:0001/mmc1:0001:1'
    option htmode 'VHT80'
    option country 'US'

config wifi-iface 'default_radio0'
    option device 'radio0'
    option network 'lan'
    option mode 'ap'
    option ssid 'TravelRouter'
    option encryption 'psk2'
    option key 'yourpassword'
