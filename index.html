Building a Travel Router with Raspberry Pi

By Your Name June 15, 2023 15 min read

My complete travel router setup using a Raspberry Pi 4

Have you ever been frustrated with hotel Wi-Fi that limits connections to one device? Or wanted to create a secure network while traveling? In this tutorial, I'll walk you through how I built a portable travel router using a Raspberry Pi that solved these problems and more.

This guide will take you through the entire process from gathering components to configuring the software and using your router on the go.

What You'll Need

  • Raspberry Pi (I used a Pi 4, but Pi 3B+ works too)
  • MicroSD card (16GB or larger recommended)
  • USB Wi-Fi adapter (for dual Wi-Fi capability)
  • Portable power bank (optional but recommended)
  • Small case for protection
  • Ethernet cable (optional)

Software Setup

We'll be using OpenWrt as our router operating system. It's lightweight, powerful, and perfect for this application.

Step 1: Download and Flash OpenWrt

First, download the appropriate OpenWrt image for your Raspberry Pi model from the official OpenWrt site. 

# Flash the image to your SD card
sudo dd if=openwrt-bcm27xx-bcm2711-rpi-4-squashfs-factory.img of=/dev/sdX bs=4M conv=fsync
# Replace /dev/sdX with your SD card device

After flashing, insert the SD card into your Raspberry Pi and connect it to your computer via Ethernet.

Basic Configuration

Once OpenWrt is running, you'll need to access the web interface to configure it.

Step 2: Access the Web Interface

Connect to your Raspberry Pi using SSH or by navigating to 192.168.1.1 in your web browser. 

ssh root@192.168.1.1
# Default password is usually blank

Wi-Fi Configuration

Now we'll set up the dual Wi-Fi configuration - one interface to connect to an existing network, and one to create our own network.

Step 3: Configure Wi-Fi Interfaces

In the OpenWrt interface, navigate to Network → Wireless and configure both interfaces:

  • The built-in Wi-Fi: Set as an access point (creates your network)
  • USB Wi-Fi adapter: Set as a client (connects to hotel/external Wi-Fi)
# Example configuration in /etc/config/wireless
config wifi-device 'radio0'
    option type 'mac80211'
    option channel '36'
    option hwmode '11a'
    option path 'platform/soc/fe300000.mmcnr/mmc_host/mmc1/mmc1:0001/mmc1:0001:1'
    option htmode 'VHT80'
    option country 'US'

config wifi-iface 'default_radio0'
    option device 'radio0'
    option network 'lan'
    option mode 'ap'
    option ssid 'TravelRouter'
    option encryption 'psk2'
    option key 'yourpassword'
